# @quarks-js/hr


<br/>

### Hr Quark

Styled Horizontal Rule react component.

<br/>

## Installation

```sh
npm install @quarks-js/hr --save
```

<br/>

## Setup
Install the package and use ```import``` statement in your application

```tsx
import { Hr } from "@quarks-js/hr"
```

<br/>

## Basic Usage
Your ```jsx|tsx``` file:
```jsx
import { Hr } from "@quarks-js/hr"

function Component() {
    return (
        <div>
            <div>{/* Some content */}</div>
            
            <Hr/>
            
            <div>{/* Some content */}</div>
        </div>
    ) 
}
```
